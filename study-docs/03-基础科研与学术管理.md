## 一、科学方法论：作者的认识论基础

### 1.1 科学定义的三层演进

源自 `108908650.html`《什么是科学？》。

**第一层：Popper 的可证伪性（Falsifiability）**
- Karl Popper 于 1930 年代提出
- 科学的本质在于"可被实验驳斥"
- 真科学必须能被证伪，伪科学则不能
- **应用**：作者批判超弦理论的核心武器——超弦拥有 10^500 个自由度（可能的宇宙），任何实验结果都可以被某个版本的宇宙所"解释"，因此从根本上不可证伪，是典型伪科学

**第二层：Kuhn 的范式理论（Paradigm）**
- Thomas Kuhn 于 1950 年代提出
- 科学由互相竞争的理论体系（范式）构成
- 科学进步表现为范式更替，而非单纯的实验累积

**第三层：Lakatos 的真伪分辨准据（Demarcation Criteria）**
- Imre Lakatos 综合 Popper 和 Kuhn
- **进步学术**：能预测意外的实验结果
- **退步学术**：只能事后解释，且不断放松自由度以自圆其说
- **应用**：作者认为超弦理论、暴涨理论都属于退步学术——一开始是合理假设，但被证伪后不是被抛弃，而是放松自由度到近乎无限以便自圆其说，本应扔进垃圾堆却被几万名教授当作宝贝继续研究

### 1.2 作者的方法论原则

| 原则 | 内容 |
|------|------|
| 事实与逻辑 | 博客座右铭，事实优先于逻辑。论证顺序严格遵循"事实→逻辑→结论→立场" |
| Occam's Razor | 奥卡姆剃刀，与可证伪性并列的科学基石 |
| Russell's Teapot | 提出方案方有责任证明其有事实逻辑基础，而非要求反方提供数学级绝对证明 |
| First Principles | 从基本原理推演，而非依赖类比联想。强调"类比联想不是逻辑，因果关系才是" |
| 数量级估算 | 对技术成功概率以 99%/90%/50%/10%/1%/0.1% 分级，只求量级精确 |
| 信息论视角 | 科学是压缩信息的方法，好理论用少量假设解释大量现象 |
| 长期复盘 | 每文多附"后注"，跨度数年持续验证预言 |

### 1.3 对美式经济学的否定

> 美式经济学连伪科学都谈不上，因为它既不可证伪，也没有预测能力。

## 二、量子力学诠释批判

### 2.1 Copenhagen 解释的逻辑漏洞

源自 `108908797.html`–`108908799.html`《谈量子力学》三部曲。

- Bohr 的 Copenhagen 解释主张"波函数塌缩"——测量时波函数瞬间塌缩到某个本征态
- **逻辑漏洞**：测量仪器和被测粒子同属一个量子波，凭什么假设测量仪器是"经典的"？

### 2.2 量子去相干是逻辑正解

源自 `108908828.html`《量子去相干详解》。

- Sidney Coleman 在 Harvard 推动的量子去相干理论
- 整个宇宙只有一个波函数
- 测量过程是"被测粒子+测量仪器"形成纠缠态（Mixed States）而非纯态
- 时间方向由量子去相干决定
- 这打破了 Copenhagen 解释的"塌缩"谬误

### 2.3 Bohmian Mechanics 是唯一逻辑自洽的解释

- de Broglie 提出、David Bohm 于 1952 年建构完成的导航波理论（Bohmian Mechanics, BM）
- 是唯一逻辑严密完整且明确标定因果关系的量子力学解释
- 但 BM 是非局部的，违反相对论

### 2.4 Many-Worlds Interpretation 的批评

- Hugh Everett 1957 年提出的多世界解释违反 Occam's Razor
- 超弦论者偏爱多重世界解释，因为同样不可证伪

### 2.5 Bell 不等式与非局部性

- John Stewart Bell 于 1964 年提出 Bell 不等式
- 实验证明量子力学是 Irreducibly Non-local（不可约非局部）
- Einstein 的"God does not play dice"和"God does not use telepathic methods"都被推翻
- 相对论的局部性假设被推翻
- 但量子力学与相对论的不相容性要到 Planck Scale（约 10^19 GeV）才明显化

### 2.6 von Neumann 的错误证明

- von Neumann 曾"证明"Hidden Variables 理论不可能
- 但证明有错，阻碍了 BM 发展多年

### 2.7 关键概念

- **Configuration Space vs Physical Space**：量子力学立足 Configuration Space，量子场论立足 Physical Space，这是 BM 与量子场论不相容的根源
- **Mixed States**：测量过程形成纠缠态而非纯态
- **薛定谔的猫**：Copenhagen 解释引发的悖论，也催生了 New Age 伪科学

## 三、高能物理批判

### 3.1 标准模型后的停滞

源自 `108908796.html`《高能物理的绝唱（二）》、`108908804.html`《从暗物质谈起》。

- 标准模型于 1974 年完成，至今 40 多年无突破
- LHC 发现 Higgs 玻色子但未发现超对称粒子
- 750GeV 统计鼓包消失
- 超对称被 LHC 和 LUX/PandaX 实验彻底否定

### 3.2 高能物理是"行尸走肉式的存在"

- 高能物理已无新发现
- 但全球几万名教授仍靠它吃饭
- 于是转向科幻式理论（超弦、暴涨）和无意义的大对撞机游说

### 3.3 暗物质实验的否定性结果

- PandaX 和 LUX 实验结果否定超对称暗物质
- 悟空卫星的"疑似发现"只是统计噪音（Look Elsewhere Effect 导致统计误差被严重低估）
- AMS-02 实验（丁肇中领导）与暗物质无关

### 3.4 ILC 优于环形对撞机

- 如果一定要建对撞机，国际直线对撞机（ILC）比环形对撞机便宜且有意义
- 但即便如此也不值得中国投资

### 3.5 关键概念

- **SSC 被裁**：美国 1993 年裁掉超导超级对撞机，是高能物理衰落的标志
- **750GeV 统计鼓包**：2015 年 LHC 出现的统计鼓包，被高能物理界炒作，2016 年消失，作者预测其可能性在 1% 以下
- **Look Elsewhere Effect**：统计误差被低估的头号罪魁祸首，假设报告的实验是全世界唯一一次类似实验，实际上同行有几千人考虑过天文量级的理论可能

## 四、"牛屎文化"概念

### 4.1 定义

源自 `108908639.html`《丁肇中与高能物理界的牛屎文化》、`108908820.html`《张首晟和凝态物理界的牛屎文化》。

**"牛屎文化"（Bullshit Culture）**：以谎言哄骗自己或别人的学术风气，源自超弦理论。

作者引用 Harry Frankfurt 的哲学概念"Bullshit"——比谎言更恶劣，因为谎言还关心真相，牛屎则完全不在乎真假。

### 4.2 起源：超弦理论

- 超弦理论起源于 Witten 1984 年的工作
- Peter Woit 的"Not Even Wrong"博客是批判超弦的先驱
- 超弦 10^500 个宇宙使其成为典型伪科学
- 但全球几万名教授靠它吃饭，于是发展出各式各样的借口涂脂抹粉、对外推销

### 4.3 扩展到凝态物理：张首晟"天使粒子"

- 张首晟团队的"天使粒子"（Majorana Fermion）是典型的"追救护车"研究
- 真正创始人是 Kane & Mele 2005 年的工作
- 张首晟还开设丹华资本圈钱
- 2022 年论文被撤稿

### 4.4 扩展到其他领域

牛屎文化已扩展到：
- 医学（89% 论文无法复制）
- 心理学
- 量子计算
- 多个领域

### 4.5 与"追救护车"的关系

**"追救护车"（Ambulance Chasing）**：法律界术语，指律师追逐救护车找客户。作者借用此概念批评跟进流行话题的延伸性研究，这与"Herd Mentality"（兽群心态）相关。

### 4.6 关键概念

| 概念 | 定义 |
|------|------|
| 牛屎文化 | 以谎言哄骗自己或别人的学术风气 |
| Ambulance Chasing | 追救护车，跟进流行话题的延伸性研究 |
| Herd Mentality | 兽群心态，学术界一窝蜂追逐热门话题 |
| Replicability Crisis | 可复制性危机，70-89% 论文无法复制 |
| 暴涨理论也是牛屎 | 一开始是合理假设，但被证伪后放松自由度到近乎无限以自圆其说 |

|------|------|----------|

## 六、具体科研项目评价

### 6.1 大对撞机（CEPC）

源自 `125380740.html`《大对撞机不是好的基础科研项目》。

**反对理由**：
1. **无理论基础**：探索的是能量级别高于 LHC 一个数量级的全未知物理，根本谈不上实用价值，且实际上并没有任何理由必须存在新物理
2. **性价比极低**：投资千亿级别，预期产出为零
3. **美国已放弃 SSC**：1993 年裁掉，是前车之鉴
4. **丘成桐公然撒谎**：说秦皇岛对撞机为 Higgs 设计，实际上 Higgs 已被 LHC 发现
5. **王贻芳的 Strawman Attack**：采用稻草人攻击，歪曲反对者的论点

**关键事件**：
- 据何祚麻院士披露，是杨振宁先生亲自写信给习近平反对对撞机
- 领导人做了批示，中科院成立 11 人委员会审查
- 最后以 6 比 5 的微弱投票优势否决

### 6.2 量子通信（QKD）

源自 `171633910.html`《量子通信和计算是中国学术管理的头号误区》。

**核心论点**：
1. **量子通信是"假未来科技"**：唯一价值在于抵抗量子计算破密，但后量子加密法已使这一价值归零
2. **Shor's Algorithm 与 RSA**：Shor's Algorithm（1994 年发明）能破解 RSA 加密，是量子计算唯一真正的威胁。但 30 年努力、几千亿美元投资，RSA 被简单廉价地替换掉了
3. **后量子加密法已解决问题**：
   - NTRU（1996 年）
   - LWE（2005 年）
   - NIST 2024 年正式采纳后量子密码标准
   - CACR（中国密码学会）2020 年已推荐后量子加密法，但被科技部忽视
4. **NSA 公开评估量子通信无用**：但中国信息学会对 NSA 报告的翻译做了多处篡改
5. **工程上无法解决**：QKD 有诸多工程难题——中继器可能被攻击、身份确认问题、DOS 攻击无解等
6. **军队早就弃用 QKD**：行内人透露
7. **潘建伟量子通信公司已彻底烂尾**：四五年前就彻底烂尾，但不追责，连媒体讨论都被全面禁止，比当年汉芯的陈进还要离谱得多

### 6.3 量子计算

**核心论点**：
1. **距实用差 8 个数量级**：如果幸运的话，60 年后可能有真正的应用
2. **"量子霸权"是先射箭再画靶**：挑选无实际用处但特别方便量子计算得结果的任务，故意不优化传统电脑程序，人为创造虚伪的比较优势
3. **玻色子采样（Boson Sampling）**：量子计算用来吹嘘"量子霸权"的任务，无实际用途
4. **潘建伟照抄博导实验设计**：在科学上不值得花大钱
5. **中科大全面封锁媒体**：潘建伟集团有整个中科大为靠山，对内宣管道的掌控近乎绝对
6. **量子计算是金融诈骗**：第一代老鼠会成员将无意义、无价值的"货品"高价卖给第二代和第三代会员
7. **深圳一地就有几万个量子计算专利**：每一个都会有人试图拿来做金融诈骗

### 6.4 核聚变发电

**核心论点**：
1. **Tokamak 设计成功概率 10^-10**，Stellarator 为 10^-5
2. **高能中子（14MeV）轰击内腔壁**，无材料可承受 60 年
3. **等离子体 MHD 不稳定性**，容错裕度小于 1 秒（核裂变为 1 天）
4. **反应器寿命可能短至 6 个月**，需像 Chernobyl 一样水泥封存
5. **氚在地球无可开采存量**，必须人工合成
6. **"谎言越大越难揭发"**（Goebbels 语），每代实验周期 30 年，真相大白时研究者已死光

**评审必须跨行业**：
- 核污染专业评审 Tritium 问题
- 材料学专家审查内壁结构
- 核裂变项目主管做经济比较
- 核工程总师估算时程

### 6.5 AI

**核心论点**：
1. **2012 年突破来自 GPU 算力**：AI 在 2012 年的突破来自 GPU 算力终于达到神经网络所需的基本程度
2. **NN 的极限**：神经网络在象棋、围棋等每一步只有少数逻辑可能的问题上效率很高，但遇到开车这样有真正近乎无限可能遭遇的环境，就无法完美处理每一个小概率事件
3. **大数据分析的共通弱点**：现实人生中总有统计数据没有涵盖的真正黑天鹅，任何基于对经验来做非线性归纳的研究工具先天不可能做出合适反应
4. **蛋白质折叠可能像围棋**：蛋白质折叠可能比较像围棋（有良好定义的目标），所以 AI 可能有突破
5. **AI 不是万灵丹**：市场经济和国际竞争会自然吹捧过度

### 6.6 美国 Sematech 成功案例

- 1980 年代美国半导体产业面对日本竞争
- 政府组织 Sematech 联合研发
- 成功扭转局势
- 这是政府正确介入战略产业的范例

## 八、知识脉络总览

```
一、科学方法论基础

二、量子力学诠释
Copenhagen 解释有逻辑漏洞 → 量子去相干是正解
→ Bohmian Mechanics 是唯一逻辑自洽的解释
→ Bell 不等式证明量子非局部性 → 相对论局部性假设被推翻

→ LHC 发现 Higgs 但无超对称 → 750GeV 鼓包消失
→ 高能物理成为"行尸走肉" → 转向大对撞机游说

四、牛屎文化扩展
超弦（1984 年 Witten）→ 暴涨理论 → 张首晟天使粒子
→ 量子通信 → 量子计算 → 医学（89% 论文无法复制）

五、中国学术管理
文革后矫枉过正 → 学术界获得极大自治 → 学阀体系形成
→ 与资本、地方政府勾结 → 造假夸大无代价 → 学术文化糜烂
→ 双循环战略提出新要求 → 必须由政府出手整顿

六、具体项目评价
大对撞机（无理论基础、性价比极低、杨振宁反对成功）
→ 量子通信（后量子加密法已解决、工程无法实现、烂尾不追责）
→ 核聚变（Tritium 问题无解）
→ 量子计算（距实用差 8 个数量级、金融诈骗）
→ AI（2012 年突破、有极限、不是万灵丹）
```
