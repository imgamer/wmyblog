## 一、方法论基础：事实与逻辑

### 1.1 核心立场

源自 `124083467.html`《为什么事实与逻辑对群众无效？》、`108908784.html`《如何辨认谎言》、`131023197.html`《常见的狡辩术》。

- 追求真相只有逻辑和实验是可靠准绳
- 多数人、权威、大众媒体都是非常不稳妥的依据
- 引入 Dunning-Kruger 效应：能力越低者越倾向于高估自己，形成"笨蛋峰"（Mt. Stupid）
- 民选制的基本假设（选民理性、能辨别是非）被证伪

### 1.2 逻辑思维 vs 感性思维

- **逻辑思维**：基于事实与推演
- **感性思维**：基于情绪与立场
- 大多数人在大多数时候使用感性思维

### 1.3 测谎的科学结论

- 依靠表情、紧张程度判断谎言"与掷骰子随机猜测没有足够差别"
- 正确方法是 Ormerod 提出的"忽略表情、专注答案细节、寻找自相矛盾"

### 1.4 狡辩术分类

源自 `131023197.html`《常见的狡辩术》。

| 狡辩术 | 定义 |
|--------|------|
| 移花接木（Straw Man） | 歪曲对方论点然后攻击歪曲后的版本 |
| 人身攻击（Ad Hominem） | 攻击对方人格而非论点 |
| 鱼目混珠 | 把联想/相关误为因果 |
| 浑水摸鱼 | 利用语义学不确定性 |

## 二、美式民主与富豪政治批判

### 2.1 财阀反动的历史脉络

源自 `108908682.html`《富豪口袋里的国家》、`116683072.html`《美国政坛的系统性腐化》。

**四层因果结构**：

1. **第一层（表象）**：最富千分之一/万分之一人口所占财富比率创历史新高
2. **第二层（政策）**：新政与大社会被逐步反转（Rolled Back），连民主党总统也追求"自由化"
3. **第三层（根源）**：贪腐合法化与游说工业的建立，使政策主控权落入富豪阶级
4. **第四层（逻辑）**：宣传战炮口对内，通过六大传媒集团（GE、Fox、Disney、Viacom、Time-Warner、CBS）控制舆论

### 2.2 关键历史节点

| 时间 | 事件 |
|------|------|
| 1929 | 大萧条 |
| 1933 | 罗斯福新政 |
| 1940-60 年代 | 全盛期（最富千分之一财富占比降至 8% 以下） |
| 1970 年代 | 财阀大反扑 |
| 1980 | Reagan 入主白宫 |
| 2008 | 金融危机 → Dodd-Frank 法案 |
| 2014 | 共和党废除 Section 716（花旗银行代笔条款），使整个法案形同虚设 |

### 2.3 关键概念

- **"美式金主贪腐"与"政府官员主控贪腐"的本质区别**：前者绝对不平等且完全合法，无改善机制；后者非法，可依法铲除
- **"美式民主的真谛：主子的确是民间的，只不过不是全民，而是极少数的特权阶级"**

## 三、金融机制作为财阀统治的工具

源自 `108908689.html`《美式民主的真正主人》。

### 3.1 金融官僚体系的叠床架屋

- 联储会、SEC、CFTC 各有职权
- 结果是"大家都有权管，一旦出事没人有责任"

### 3.2 高频交易（HFT）的掠夺

- 每年从大投资人挤出近 100 亿美元利润
- 是共同基金长期 underperform 的主因
- 最终吃亏的是小投资人
- Stiglitz 因主张对 HFT 征税被踢出 SEC 专家小组
- 作者本人曾开发世界第一个全自动交易程序（2001 年为 UBS 所创，达总交易量 5%），2003 年被禁

### 3.3 高盛对 SEC 的掌控

- "到了可笑的地步"
- Carmen Segarra 事件证明即使公布录音也无法撼动体系

### 3.4 爱因斯坦名言

> 只有两件事情是无限的，宇宙和人类的愚蠢。

## 四、言论自由的假相

源自 `108908692.html`《言论自由的假相》。

### 4.1 核心论证

- 美国宣传战的逻辑前提是"美国总统等同于其他国家领导阶层"
- 但实际上"真正的权力中心自尼克森后根本不在政府，而在幕后的金主，总统只是傀儡木偶，也是箭靶"
- 对总统的批评怎么离谱都可以（如出生在国外、是外国间谍）
- **但"就是不能把他和金主之间的利益交换说明白了"**

### 4.2 案例

- 作者在《时代周刊》发短评指出奥巴马不敢得罪金主，不到一小时被删
- 而谩骂奥巴马是假美国人的疯狂评论一直留下
- CNN 资深主播 Jim Clancy 因称两个 Twitter 账号是"以色列的公关 PR 而不是人权 HR"被开除
- Mel Gibson 因酒驾时说出"以色列是侵略者"而演艺生涯完蛋

### 4.3 核心结论

> 美国对内部言论的控制其实比中共更严格，对有异见者的惩罚也更加彻底。

## 五、美国的开国神话

源自 `108908793.html`《美国的开国神话》。

### 5.1 神话的解构

- 美国独立战争并非"自由对抗暴政"的浪漫叙事
- 华盛顿等开国元勋多为大奴隶主、大地产主
- 独立的真正动力是西部土地利益和税收分配
- "无代表不纳税"的真正诉求是殖民地议会要求与英国议会平起平坐

### 5.2 神话的功能

- 为美国例外论提供历史根基
- 为输出"民主"提供道德正当性
- 掩盖美国建国即财阀政治的实质

源自 `116683072.html`《美国政坛的系统性腐化》。

### 6.1 合法贪腐的机制

## 七、现代英美的假新闻体系

源自 `131371444.html`《现代英美的假新闻体系》。

### 7.1 假新闻的层级

1. **直接造假**：无中生有的报道
2. **选择性报道**：只报道符合叙事的事实
3. **框架操控**：用特定框架解读事实
4. **议题设定**：决定哪些议题值得报道
5. **信源操控**：培养"专家"提供特定观点

### 7.2 六大传媒集团

- GE
- Fox
- Disney
- Viacom
- Time-Warner
- CBS

这六大集团控制了美国绝大多数主流媒体，形成寡头垄断。

## 八、如何破解当前欧美的宣传攻势

源自 `162032391.html`《如何破解当前欧美的宣传攻势》。

### 8.1 宣传战的本质

- 宣传战是第三世代殖民帝国的核心统治手段
- 美国宣传战的目的是维持美元霸权
- 中国是被抹黑的头号目标

### 8.2 破解策略

1. **建立独立的信息渠道**：不能依赖西方通讯社
2. **培养批判性思维**：教育公众识别宣传手法
3. **主动设置议题**：不能只被动回应
4. **利用对方内部矛盾**：西方并非铁板一块
5. **用事实和逻辑反击**：长期坚持"事实与逻辑"

## 九、科学与民主

源自 `108908693.html`《科学与民主》。

### 9.1 核心论点

- 科学和民主是两个独立的概念
- 科学基于事实和实验，民主基于多数决
- 多数决不能决定科学真伪
- 美式"民主=科学=真理"的三位一体是宣传话术

### 9.2 民主的局限

- 选民理性假设被 Dunning-Kruger 效应证伪
- 多数人的意见不等于真理
- 民主需要前提：教育水平、信息透明、理性讨论

## 十、冷战、学运和五四运动

源自 `164999328.html`《冷战、学运和五四运动》。

### 10.1 冷战的本质

- 冷战并非"自由对抗极权"的意识形态战争
- 本质是美苏两个第三世代殖民帝国的争霸
- 美国通过宣传将争霸包装为意识形态对决

### 10.2 学运的操控

- 学运看似自发，实则背后有组织力量
- CIA 通过 NED 等组织资助海外学运
- 香港占中、台灣太阳花都有外部势力痕迹

### 10.3 五四运动的再评价

- 五四运动有反帝反封建的进步意义
- 但"德先生"（民主）和"赛先生"（科学）的并列是误导
- 科学不依赖民主，民主也不必然产生科学

源自 `150052301.html`《欧系文明的起源》。

### 11.1 文明起源的唯物史观

- 欧系文明起源于古希腊

## 十四、大众媒体的内建矛盾

源自 `111157635.html`《大众媒体的内建矛盾》。

### 14.1 内建矛盾

- 媒体自称"第四权"监督政府
- 但媒体本身是企业，受利润驱动
- 利润驱动导致煽情化、娱乐化
- 与"严肃新闻"的使命矛盾

### 14.2 解决方案

- 公营媒体（如 BBC 模式）有局限
- 完全市场化媒体必然堕落
- 需要制度创新
